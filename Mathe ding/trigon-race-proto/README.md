# Trigon Race — Frontend Prototype

This is a client-only prototype demonstrating the Teacher and Student views, triangle construction canvas, and local validation of congruence constraints.

Quick start:

```bash
cd "./trigon-race-proto"
npm install
npm run dev
```

Open the app in your browser (Vite will print the dev URL). Switch between Teacher and Student views and try dragging vertices and submitting constructions.

Notes:
- All game state is simulated client-side; no backend required.
- Math tolerance is ±1% by default.
